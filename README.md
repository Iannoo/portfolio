## ğŸš¦ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Iannoo/portfolio.git

```

2. Install dependencies

3. Start development server
```bash
npm run dev
```

4. Build for production
